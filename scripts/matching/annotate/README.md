# LLM
To run LLM - Ground:
```
./run_llm_ground.sh
```

To run LLM - Noisy:
```
./run_llm_noisy.sh
```

To run LLM - Multi-Teacher:
```
./run_llm_multi.sh
```

# SLM
To run SLM - Ground:
```
./run_slm_ground.sh
```

To run SLM - Noisy with different LLM Annotators:
```
./run_slm_noisy.sh
```

To run SLM - Noisy with different Training Size:
```
./run_slm_noisy_size.sh
```
