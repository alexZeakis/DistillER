To run fine-tuning on ground-truth data, run:
```
cd scripts/plm/
./run_finetuning_ground.sh
```

To run fine-tuning on noisy data, run:
```
cd scripts/plm/
./run_finetuning_noisy.sh
```

To run fine-tuning on ground-truth data with down-sampling, run:
```
cd scripts/plm/
./run_finetuning_ground_ds.sh
```

To run fine-tuning on noisy data with down-sampling, run:
```
cd scripts/plm/
./run_finetuning_noisy_ds.sh
```
