# GRPO
To run rl with GRPO on SLM and LLM annotators:
```
./grpo/run_grpo.sh
```

To run rl with GRPO on SFT model:
```
./grpo/run_grpo_SFT.sh
```

To run rl with GRPO with different epochs:
```
./grpo/run_grpo_epochs.sh
```
